# Docker final task

## instruction

1. clone repo and cd into project app
   ```
   git clone git@github.com:ngga-wt/docker_final_task.git && cd docker_final_task
   ```
2. make executable to run-docker.sh file
   ```
   sudo chmod +x run-docker.sh
   ```
3. execute run-docker.sh file
   ```
   ./run-docker.sh
   ```
4. in order to test if python entrypoint file is keep running after 10 seconds
   you shuld pass argument to run-docker.sh file
   ```
   ./run-docker.sh 15
   ```
